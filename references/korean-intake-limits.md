# Korean Intake Limits (KFDA I2710 큐레이션)

> 식약처 I2710 "품목분류 정보" API가 제공하는 **건강기능식품 원료별 1일 섭취량 기준**(522종) 중, `rda-table.md`에 수록된 28개 핵심 영양소만 선별한 참조 테이블.
>
> **용도**: 한국 공식 섭취 상한·하한이 NIH RDA/UL과 다를 때 리포트에 병기. NIH 기준이 없고 KDRI만 있을 때의 대체 기준.

---

## 1. 기준 정의

### 1-1. DAY_INTK_LOWLIMIT / DAY_INTK_HIGHLIMIT (I2710 필드)
- **하한(LOWLIMIT)**: 건강기능식품으로 인정받기 위한 **최소 1일 섭취량** (기능성 발현 기준)
- **상한(HIGHLIMIT)**: 안전하게 섭취 가능한 **최대 1일 섭취량** (식약처 UL에 준함)
- 출처: 식품의약품안전처 「건강기능식품의 기준 및 규격」

### 1-2. NIH UL와의 차이 원칙
- **원문 병기** — KDRI(한국 기준)과 NIH(미국 기준)이 다를 때 양쪽 모두 표시
- 보수적 판단 — 두 기준 중 **낮은 상한** 기준으로 UL 초과 경고
- 사용자가 한국 거주자면 KDRI/KFDA 우선 안내

### 1-3. 단위
- KFDA 원문의 `INTK_UNIT`을 그대로 표기 (mg·µg·g·IU)
- 변환 시 `rda-table.md`의 단위 변환식 참조

### 특이사항
- **비타민 A·D·E·B6**: KDRI가 NIH UL보다 엄격. 한국 거주자는 KDRI 기준으로 UL 초과 판정 권장
- **비타민 D**: I2710 원문은 µg 단위. IU 환산 시 1µg = 40IU → 상한 25µg = **1000IU**. NIH UL 100µg = 4000IU와 4배 차이
- **나이아신**: 니코틴산(niacin) vs 니코틴아미드(niacinamide) 구분. UL 35mg은 니코틴산 기준

### 특이사항
- **마그네슘**: 양쪽 350mg UL 모두 **보충제에서만** 기준. 식이 포함하면 400mg 이상 섭취 흔함 → 경고 표시 시 "보충제 기준" 명시

## 6. 사용 방법 (Claude 내부 참조)

### 6-1. RDA/UL 평가 시 (analyze-ingredient.md에서 호출)
```
1. rda-table.md에서 NIH RDA/UL 확인
2. 본 문서(korean-intake-limits.md)에서 KDRI/KFDA 기준 확인
3. 두 기준이 다르면:
   - 리포트에 양쪽 수치 병기: "NIH UL 100µg / 식약처 상한 25µg"
   - UL 초과 판정은 **둘 중 낮은 값** 기준 (보수적)
4. NIH 기준이 없고 KDRI만 있으면 KDRI 기준으로 판정 + "한국 공식 기준" 명시
```

### 6-2. 리포트 템플릿에서 (report-template.md)
```markdown
> **비타민 D**: 일일 섭취량 125µg (5000IU)
> - 식약처 상한 25µg (1000IU) 대비 **5배 초과** — 한국 기준 UL 초과
> - NIH UL 100µg (4000IU) 대비 125% — 미국 기준도 초과
> - [출처: Tier 1 / NIH ODS] + [출처: Tier 2 / 식약처 KDRI 2020]
```

### 6-3. I2710 실시간 조회가 필요한 경우
```
조건: 본 문서에 없는 기능성 원료(§5 외)의 정확한 섭취 기준 확인 필요
엔드포인트: kfda-htfs-api.md §6 참조
예: http://openapi.foodsafetykorea.go.kr/api/{KEY}/I2710/json/1/5/PRDCT_NM=밀크씨슬
```

## 8. 한계

- I2710은 건강기능식품 **원료 기준**. 일반식품에서의 섭취량은 별도 기준(KDRI 식사섭취기준) 적용
- I2710 상한은 "보충제로 섭취 가능한 안전 상한" — 식단 포함 총 섭취량 UL과 다를 수 있음
- 기능성 원료는 추출·가공 방법에 따라 생물학적 활성이 크게 달라짐 → 제품별 개별 판단 필요
- 본 테이블은 큐레이션본. 전체 522종은 I2710 API로 실시간 조회
