## 1. 비타민 A (Retinol / RAE)

### 1.1. 비타민 A 부족 시
- **초기**: 야맹증 (암순응 지연), 각막 건조
- **중기**: 각막 연화증, 피부 각화, 감염 취약성 ↑ (특히 호흡기·설사)
- **장기/중증**: 실명 (세계보건기구: 개도국 주요 실명 원인), 면역 저하로 사망률 ↑
- **고위험군**: 흡수장애(지방변), 만성 알코올, 극단적 저지방 식단, 낭포성섬유증
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminA-HealthProfessional/]

### 1.2. 비타민 A 초과 시
- **단기 (급성, 수십만 IU 일시 섭취)**: 두통, 구역, 현기증, 피부 박리
- **장기 (만성, UL 3,000µg RAE/일 초과 수 개월)**:
  - 간독성, 고칼슘혈증, 두개내압 상승 (위가뇌종양)
  - **골절 위험 ↑** — 고령 여성에서 고용량 retinol이 대퇴골 골밀도 ↓
  - **기형 유발 (Teratogenic)** — 임신 초기 10,000 IU 이상 섭취 시 중추신경 기형 위험
- **위험 임계값**: 성인 UL 3,000 µg RAE/일 (10,000 IU). 임신부는 더 엄격
- **주의**: β-카로틴(식물성)은 체내 전환 조절되어 독성 ↓. 동물성 retinol은 축적됨
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminA-HealthProfessional/]

## 2. 비타민 D

### 2.1. 비타민 D 부족 시
- **초기 (25(OH)D 20-30 ng/mL, 경계)**: 무증상, 뼈대사 마커 이상 시작
- **중기 (12-20 ng/mL, 부족)**: 근력 저하, 낙상·골절 ↑, 면역 약화, 만성 피로
- **장기/중증 (<12 ng/mL, 결핍)**:
  - **성인**: 골연화증(osteomalacia), 이차성 부갑상선 항진 → 골다공증 가속
  - **아동**: 구루병 (뼈 변형, 성장 지연)
  - 심혈관 질환·자가면역질환·특정 암 위험 ↑ (역학적 연관)
- **고위험군**: 고위도 거주, 실내생활, 짙은 피부, 비만(지방조직에 격리), 흡수장애, 고령
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminD-HealthProfessional/]

### 2.2. 비타민 D 초과 시
- **단기 (>10,000 IU/일 수 일~수 주)**: 오심, 구토, 식욕부진, 변비, 갈증
- **장기 (UL 4,000 IU/일 초과 수 개월+)**:
  - **고칼슘혈증** → 신결석, 신석회화
  - **심혈관 석회화** (관상동맥·판막), 부정맥
  - **정신 증상**: 혼돈, 기면
  - **역설적 골흡수 ↑**: 매우 고용량에서 뼈 재흡수 증가
- **위험 임계값**: 혈중 25(OH)D > 100 ng/mL에서 독성 증거, > 150 ng/mL에서 명확한 증상
- **고위험군**: 사코이도시스·결핵·림프종(활성형 1,25(OH)2D 과생산), 칼슘 보충제 병용
- **약물 상호작용**: 디곡신(고칼슘혈증 시 독성 ↑), 티아지드 이뇨제(칼슘 축적)
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminD-HealthProfessional/]

## 3. 비타민 E (α-Tocopherol)

### 3.1. 비타민 E 부족 시
- **원인**: 거의 없음 (일반 식단으로 충분). 흡수장애 시만 발생
- **증상**: 말초신경병증, 근병증, 망막증, 면역 약화
- **고위험군**: 지방흡수장애 (낭포성섬유증·abetalipoproteinemia·담즙정체)
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminE-HealthProfessional/]

### 3.2. 비타민 E 초과 시
- **단기**: 경미 — 구역, 설사
- **장기 (UL 1,000 mg/일 α-TE)**:
  - **출혈 위험 ↑** (혈소판 응집 억제) — 특히 와파린·아스피린 병용 시 위험
  - 일부 메타분석: 400 IU/일 이상 장기 복용 시 총사망률 소폭 ↑ 가능성 제기 (2005 Miller et al.)
- **위험 임계값**: 성인 UL 540 mg α-TE (≈ 800 IU 합성 / 540 IU 천연)
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminE-HealthProfessional/]

## 4. 비타민 K

### 4.1. 비타민 K 부족 시
- **증상**: 출혈 경향 (코피, 잇몸, 상처 지혈 지연), 멍
- **신생아**: 출혈성 질환 (신생아 비타민 K 주사 표준)
- **장기 부족 가능성**: 골다공증, 혈관 석회화 (K2 부족 가설)
- **고위험군**: 광범위 항생제 장기 투여 (장내세균 억제), 지방흡수장애, 와파린 복용
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminK-HealthProfessional/]

### 4.2. 비타민 K 초과 시
- **천연 K1/K2**: UL 설정 없음 — 과다 독성 보고 없음
- **주의**: **와파린 복용자**는 비타민 K 섭취 급변 금지 (INR 변동). 보충제 시작·중단 시 의사 상담
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminK-HealthProfessional/]

## 5. 비타민 C

### 5.1. 비타민 C 부족 시
- **초기**: 피로, 잇몸 출혈, 상처 치유 지연
- **중증 (괴혈병, scurvy)**: 콜라겐 합성 실패 → 잇몸 퇴축·치아 소실, 피부 점상출혈, 관절통, 우울
- **고위험군**: 흡연자 (요구량 +35mg/일), 신선 과채 섭취 극소, 혈액투석
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminC-HealthProfessional/]

### 5.2. 비타민 C 초과 시
- **단기 (2 g+/일)**: 설사, 복부경련, 구역
- **장기 (UL 2,000 mg/일 초과)**:
  - **신결석** (특히 옥살산칼슘) 위험 ↑ (남성·기존 결석 병력)
  - **철분 과흡수** (혈색소증 환자 악화)
  - **G6PD 결핍자**: 용혈 발작 보고
- **위험 임계값**: 성인 UL 2,000 mg/일
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminC-HealthProfessional/]

## 6. 비타민 B1 (티아민)

### 6.1. 티아민 부족 시
- **증상**: 각기병 (beriberi) — 심부전(습성 각기), 말초신경병증(건성 각기), 피로
- **중증**: 베르니케 뇌병증 → 코르사코프 증후군 (기억장애, 작화증)
- **고위험군**: 만성 알코올 (가장 흔함), 흡수장애, 신장 투석, 만성 설사, 정제 탄수화물 위주 식단
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Thiamin-HealthProfessional/]

### 6.2. 티아민 초과 시
- **UL 없음** — 수용성이라 소변으로 배출, 경구 고용량 독성 보고 거의 없음
- **주의**: 정맥주사 시 드물게 아나필락시스
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Thiamin-HealthProfessional/]

## 7. 비타민 B2 (리보플라빈)

### 7.1. 리보플라빈 부족 시
- **증상**: 구각염 (입꼬리 갈라짐), 설염, 지루성 피부염, 빈혈, 광과민
- **고위험군**: 채식·비건(유제품 없을 때), 흡수장애, 편두통 환자
- **참고**: 400mg/일이 **편두통 예방**에 효과 (Tier 1 임상시험)
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Riboflavin-HealthProfessional/]

### 7.2. 리보플라빈 초과 시
- **UL 없음** — 흡수 한계 있고 소변으로 배출
- **특징**: 고용량 시 소변이 밝은 노랑-주황색 (정상 반응, 무해)
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Riboflavin-HealthProfessional/]

## 8. 나이아신 (B3, Niacin)

### 8.1. 나이아신 부족 시
- **증상 (펠라그라, pellagra)**: "3D" — Dermatitis(피부염, 광과민), Diarrhea(설사), Dementia(치매양)
- **고위험군**: 옥수수 주식 저개발국, 만성 알코올, 카르시노이드 증후군(트립토판 소진), 하트납병
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Niacin-HealthProfessional/]

### 8.2. 나이아신 초과 시
- **니코틴산(NA) 형태**: **플러싱** (얼굴·목 홍조·화끈, 35 mg+부터 흔함), 간독성 (1g+ 장기)
- **니코틴아미드(NAD) 형태**: 플러싱 없음, 간독성 리스크 ↓, UL 더 높음
- **장기 고용량 (치료용 1.5-3 g/일)**: 간기능 이상, 당불내성, 통풍, 황반부종
- **위험 임계값**:
  - 니코틴산 UL 성인 35 mg/일
  - 니코틴아미드/나이아신 혼합 UL 1,000 mg/일 (NE 기준)
- **주의**: 스타틴과 병용 시 횡문근융해 위험 ↑
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Niacin-HealthProfessional/]

## 9. 비타민 B6 (Pyridoxine)

### 9.1. B6 부족 시
- **증상**: 구각염·설염, 말초신경병증, 우울·과민, 빈혈(소구성), 면역 저하
- **고위험군**: 만성 알코올, 신장 투석, 자가면역질환, 경구 피임약 사용, isoniazid(결핵약)·theophylline 복용
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminB6-HealthProfessional/]

### 9.2. B6 초과 시
- **주요 독성: 말초 감각신경병증** (손발 저림·통증·균형 장애)
- **장기 고용량 (100 mg/일 이상 수 개월~년)**: 가역적이나 매우 고용량(>500 mg) 오래 지속 시 비가역 가능
- **일부 보고**: 50 mg/일 수준도 장기(1년+) 복용 시 신경 증상 사례
- **위험 임계값**: 성인 UL 100 mg/일. 보수적으로 50 mg/일 이하 권장 (장기 복용자)
- **약물 상호작용**: levodopa(파킨슨약) 효과 감소 — 병용 시 의사 상담
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminB6-HealthProfessional/]

## 10. 엽산 (Folate, B9)

### 10.1. 엽산 부족 시
- **증상**: 거대적혈모구성 빈혈 (대구성), 피로, 구내염, 설사, 인지저하
- **임신 초기 부족**: **신경관 결손** (척추이분증·무뇌증) 위험 ↑
- **심혈관**: 호모시스테인 ↑ → 혈관내피 손상 (일부 역학)
- **고위험군**: 가임기 여성, 만성 알코올, 흡수장애(크론병·셀리악), 메토트렉세이트·페니토인 복용
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Folate-HealthProfessional/]

### 10.2. 엽산 초과 시
- **주요 우려**: **B12 결핍 가림 (masking)** — 엽산 보충이 거대적혈구 빈혈은 교정하지만 B12 결핍의 신경 손상은 진행
- **합성 엽산 (folic acid)**: 비대사성 혈중 엽산 축적 가능성 — 일부 연구에서 면역·인지에 영향 가설 (결정적 증거 ↓)
- **위험 임계값**: 성인 UL 1,000 µg DFE/일 (합성 기준)
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Folate-HealthProfessional/]

## 11. 비타민 B12 (Cobalamin)

### 11.1. B12 부족 시
- **혈액**: 거대적혈모구성 빈혈 (엽산 결핍과 유사)
- **신경 (B12 고유, 비가역 가능)**: 말초신경병증, 운동실조, 기억·인지 저하, 우울, 심하면 치매양
- **고위험군**: **50세 이상 (위산 저하로 흡수 ↓)**, 비건·극단 채식, 위절제술, 악성빈혈(자가면역), **메트포르민·위산억제제(PPI/H2) 장기 복용**
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminB12-HealthProfessional/]

### 11.2. B12 초과 시
- **UL 없음** — 매우 안전 (수용성, 잉여 배출)
- **주의**: 혈중 B12 매우 높으면 기저 간질환·혈액질환 신호일 수 있음 (과다 섭취가 아닌 원인 검사 필요)

## 14. 칼슘

### 14.1. 칼슘 부족 시
- **단기**: 대부분 무증상 (뼈에서 동원)
- **장기**:
  - **폐경 후 여성**: 골다공증 가속, 골절 위험 ↑
  - **아동·청소년**: 최대 골량(peak bone mass) 달성 실패
  - 근육 경련, 저림, 부정맥 (급성 저칼슘혈증)

- **고위험군**: 유제품 섭취 제한(유당불내증·비건), 폐경 후, 장기 코르티코스테로이드, 만성 신부전
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Calcium-HealthProfessional/]

### 14.2. 칼슘 초과 시
- **단기 (보충제 고용량)**: 변비, 복부팽만, 가스
- **장기 (UL 2,500 mg/일 초과)**:
  - **신결석** — 특히 보충제(식품 대비 흡수 패턴 달라짐). 식품 칼슘은 오히려 결석 ↓
  - **고칼슘혈증** → 신기능 저하, 혈관·연부조직 석회화
  - **심혈관**: 일부 메타분석에서 칼슘 보충제(+D 없이) 심근경색 위험 소폭 ↑ 가능성 (결정적 결론 없음, 논쟁 중)
  - **철·아연 흡수 저해**
- **위험 임계값**: 성인 UL 2,500 mg/일 (51세+ 2,000 mg)
- **고위험군**: 기존 결석 병력, 사코이도시스, 고용량 비타민 D 병용
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Calcium-HealthProfessional/]

- **약물 상호작용**: 비스포스포네이트·테트라사이클린·퀴놀론 흡수 저해 → 복용 간격 2시간+

## 16. 철

### 16.1. 철 부족 시
- **단계**:
  1. 저장철 고갈 (ferritin ↓) — 무증상
  2. 철결핍 적혈구조혈 — 적혈구 크기 ↓
  3. **철결핍성 빈혈** — 피로, 창백, 호흡곤란, 두근거림, 이식증(얼음·흙 갈망), 손톱 숟가락 모양, 탈모
- **아동**: 인지·학습 지연 (장기 영향 가능)
- **고위험군**: **가임기 여성(월경)**, 임신부, 헌혈자 빈번, 위절제, 크론병·셀리악, 채식(heme 철 없음)
- **저위험군**: **성인 남성·폐경 후 여성**은 부족 드묾 → 보충제 필요성 낮음
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Iron-HealthProfessional/]

### 16.2. 철 초과 시
- **단기 (급성 중독, 주로 소아 우발 섭취)**: 구토·설사(혈변), 대사성산증, 간부전, 순환쇼크
- **장기 (만성 과잉)**:
  - **헤모크로마토시스(혈색소증)** 유전자 보유자에서 급격히 악화 — 간경화·간암·당뇨·심부전·관절염·성선저하
  - 일반인: 간효소 이상, 산화스트레스, 심혈관·인슐린저항성 연관성 (역학)
  - **감염 악화** — 철은 일부 병원체의 영양원
- **위험 임계값**: 성인 UL 45 mg/일 (보충제+식품 합)
- **고위험군**: HFE 유전자 변이(혈색소증), 반복 수혈, 만성 간질환, 성인 남성(드문 철분 과잉 위험)
- **약물 상호작용**: 테트라사이클린·퀴놀론·레보티록신·비스포스포네이트 흡수 저해 → 2시간+ 간격
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Iron-HealthProfessional/]

- **경미**: 생선 트림, 속쓰림, 설사
- **고용량 (3 g+/일 EPA+DHA)**:
  - **출혈 위험 ↑** (혈소판 응집 억제) — 와파린·아스피린·항응고제 병용 시 주의
  - **심방세동 위험 소폭 ↑** 가능성 (2021 STRENGTH, REDUCE-IT 부분 분석)
  - LDL 소폭 상승 (일부 제형)
- **위험 임계값**: FDA GRAS 3,000 mg/일 (EPA+DHA). 치료용 4 g/일(icosapent ethyl)은 의사 감독
- **수은 우려**: 생선유 정제품은 수은 거의 없음. 생선 자체는 메틸수은 함량 확인(특히 임신부)
- [출처: Tier 1 / https://ods.od.nih.gov/factsheets/Omega3FattyAcids-HealthProfessional/] + [Tier 2 / https://www.heart.org/en/healthy-living/healthy-eating/eat-smart/fats/fish-and-omega-3-fatty-acids]

## 26. 코엔자임 Q10 (CoQ10)

### 26.1. CoQ10 부족 시
- **상황적 감소**: 40대 이후 체내 합성 감소, 스타틴 복용자 감소
- **증상 (결핍 자체는 드묾)**: 피로, 근약화, 근통
- **드문 유전적 결핍**: 미토콘드리아 질환
- [출처: Tier 2 / https://examine.com/supplements/coenzyme-q10/]

### 26.2. CoQ10 초과 시
- **UL 없음** — 1,200 mg/일까지 임상시험 안전 보고
- **경미 부작용**: 소화불량, 불면, 피부발진
- **주의**: 와파린 효과 소폭 감소 가능 — 병용 시 INR 모니터링
- [출처: Tier 2 / https://examine.com/supplements/coenzyme-q10/]

## 27. 실리마린 (밀크씨슬)

### 27.1. 실리마린 부족 — 해당 없음
- 필수 영양소 아님. "부족" 개념 없음.

### 27.2. 실리마린 초과 시
- **UL 없음** — 임상시험 최대 1,500 mg/일까지 안전
- **경미**: 소화불량, 설사, 두통
- **약물 상호작용**: CYP3A4·CYP2C9 약한 억제 → 일부 약물(와파린·에스트로겐·스타틴·항암제) 대사 영향 가능성 (임상적 유의성 논쟁 중)
- **알레르기**: 국화과(데이지·돼지풀) 알레르기 있는 사람 주의
- [출처: Tier 2 / https://www.nccih.nih.gov/health/milk-thistle]

## 29. 리포트 작성 시 인용 가이드

### 29.1. 부족 항목 작성 예시
```markdown
### 🟠 엽산 50% (부족 경계)

**일일 섭취**: 200 µg DFE / RDA 400 µg

**부족 시 영향**:
- 거대적혈모구성 빈혈, 피로·구내염 (장기 부족 시)
- **40대 남성 관점**: 호모시스테인 상승 → 혈관내피 손상·심혈관 질환 위험 (역학적 연관)
- 가임기 여성이 아니라 신경관 결손 리스크는 해당 없음

[출처: Tier 1 / https://ods.od.nih.gov/factsheets/Folate-HealthProfessional/]

**보완 방법**:
- 녹색잎채소(시금치 1컵 = 263µg), 콩류, 아보카도, 아스파라거스
- 종합비타민을 2정으로 복귀하거나 식단 보완
```

### 29.2. 초과 항목 작성 예시
```markdown
### 🔴 비타민 D 125 µg/일 (UL 125%)

**구성**: D3 제품 100µg + 칼슘마그네슘D 10µg + 오메가-3 10µg + 종합비타민 5µg = 125 µg

**초과 시 영향**:
- **단기 (수 주)**: 오심·식욕부진·변비·갈증 가능
- **장기 (수 개월+)**:
  - 고칼슘혈증 → **신결석·신석회화 위험**
  - 관상동맥·판막 **석회화 가속** (심혈관)
  - 매우 고용량에서 정신 혼돈·기면
- **고위험 상호작용**: 디곡신·티아지드 이뇨제 복용 시 위험 ↑ (본 사용자는 처방약 없어 완충)

[출처: Tier 1 / https://ods.od.nih.gov/factsheets/VitaminD-HealthProfessional/]

**해결 경로**:
- D3 4000IU 제품 중단 → 총 25 µg (1,000 IU) 정상화
- 혈중 25(OH)D 검사 후 결핍(<20 ng/mL) 확인 시 의사 감독 하 한시 재개
```
